## [2026.3.1](https://github.com/SukramJ/home-assistant-matter-bridge/releases/tag/v2026.3.1) (2026-03-27)

### Features

- Add auto-composed devices: automatically combine related sensors (temperature, humidity, pressure, illuminance) from the same device into a single Matter endpoint via `autoComposeSensors` feature flag
- Add per-entity device type override: override the Matter device type for specific entities via `entityOverrides` in bridge config (20 available device types)
- Add new entity domains: `water_heater` (Thermostat), `alarm_control_panel` (ModeSelect), `remote` (OnOffPlugInUnit)
- Add new filter types: `regex` (regular expression matching), `device_name` (substring match on device name), `device_class` (match on entity device class attribute)
